## Objective
Analyze e-commerce sales data to identify revenue drivers, customer behavior,
discount impact, regional performance, and sales trends.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## Key Steps
- Data cleaning and missing value handling
- Feature engineering (discount, final price, revenue)
- Exploratory Data Analysis using visualizations

## Key Insights
- Electronics category generates the highest revenue
- Returning customers contribute more revenue than new customers
- Higher discounts generally increase units sold
- North and South regions perform better
- Revenue shows stable growth over time
