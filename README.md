# data-analyst-portfolio
Project: Sales & Revenue Analysis (EDA)

Objective:
Analyze product performance, regional sales, and revenue trends.

Tools Used:
	•	Python
	•	Pandas, NumPy
	•	Matplotlib, Seaborn

Key Insights:
	•	Laptops generate the highest revenue due to premium pricing
	•	Mobiles & tablets drive higher sales volume
	•	North & South regions perform better
