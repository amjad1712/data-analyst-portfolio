# data-analyst-portfolio
Project: Student Performance Analysis (EDA)

Objective:
Analyze student performance and identify factors affecting marks using EDA.

Tools Used:
	•	Python
	•	Pandas, NumPy
	•	Matplotlib, Seaborn

Key Steps:
	•	Data cleaning & missing value handling
	•	Distribution & comparative analysis
	•	Relationship analysis & correlation

Key Insights:
	•	Attendance has a strong positive impact on marks
	•	Age shows a weak influence on performance
	•	Pass students consistently score higher
